# Guia Editorial

Esta pasta define como a Conectiva IA se comunica em todos os seus canais.

## Conteúdo

- Tom de voz
- Linguagem
- Storytelling
- Estrutura de conteúdos
- Estrutura de artigos
- Estrutura de vídeos
- Estrutura de carrosséis
- Chamadas para ação (CTA)
- Palavras recomendadas
- Palavras evitadas

Todo conteúdo produzido deverá seguir este guia.
