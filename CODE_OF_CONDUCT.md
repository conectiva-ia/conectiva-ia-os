# Código de Conduta

A Conectiva IA acredita que tecnologia deve ser construída com ética, respeito e colaboração.

Esperamos que toda interação relacionada a este projeto seja pautada por:

- Respeito.
- Transparência.
- Honestidade intelectual.
- Colaboração.
- Inclusão.
- Profissionalismo.

Não serão tolerados comportamentos ofensivos, discriminatórios ou que prejudiquem a comunidade.

Este documento poderá evoluir conforme o crescimento do ecossistema.
