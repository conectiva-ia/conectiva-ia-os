# Biblioteca de Conteúdo

Esta pasta reúne toda a estratégia e os ativos de conteúdo da Conectiva IA.

Seu objetivo é transformar conhecimento em conteúdo reutilizável para todos os canais da empresa.

Cada ativo poderá originar:

- Carrossel
- Reel
- Story
- Artigo
- Newsletter
- LinkedIn
- YouTube
- Shorts

Toda produção deve seguir os padrões definidos nesta pasta.
